## English

### What is this?

A **Claude Skill** that transforms Claude into a seasoned HR professional with 15+ years of experience. It guides you through a complete resume optimization workflow — from initial review to job-targeted tailoring — in any language.

### Features

- **Multi-language support**: Upload your resume in any language. Get optimized versions in Chinese, English, Japanese, German, French, Korean, and more — each following that market's conventions.
- **Professional HR review**: Structured feedback covering first impressions, strengths, weaknesses, ATS compatibility, and specific rewrite suggestions.
- **Job-targeted optimization**: Paste a job description, and the skill researches successful resume patterns for that role/company, then proposes targeted changes — each confirmed by you before applying.
- **Multi-position workflow**: Optimize for multiple jobs in one session. Each position gets its own tailored version.
- **Ethical by design**: Enhances and reframes your real experience — never fabricates credentials.

### Workflow

```
Step 1: Upload your resume (PDF, DOCX, TXT, or paste text)
        ↓
Step 2: Get professional review + choose output languages
        → Receive polished .docx files for each language
        ↓
Step 3: (Optional) Enter target job details
        → AI researches successful patterns
        → Proposes changes for your confirmation
        → Generates tailored .docx files
        ↓
Step 4: Repeat Step 3 for additional positions
```

### Market-Aware Formatting

| Market | Key Conventions |
|--------|----------------|
| 🇺🇸 US English | No photo, no age. 1 page for <10yr experience. Metrics-driven bullets. |
| 🇬🇧 UK English | "CV" format. 2 pages acceptable. |
| 🇨🇳 Chinese | Photo, DOB, gender expected. 1-2 pages. |
| 🇯🇵 Japanese | 履歴書 / 職務経歴書 format. Photo required. Highly structured. |
| 🇩🇪 German | Lebenslauf format. Photo, DOB common. Signed and dated. |
| 🇫🇷 French | Photo, age, nationality. "Centres d'intérêt" section valued. |
| 🇰🇷 Korean | Photo, DOB, gender. Education-first. Formal tone. |

### Installation

1. Download `resume-optimizer.skill` from this repo
2. In Claude.ai, go to **Settings → Skills**
3. Upload the `.skill` file
4. Start a new conversation and say: *"Help me optimize my resume"*

### Usage Examples

- "帮我优化一下简历" (Chinese)
- "Can you review my CV and make it better?"
- "履歴書を改善してください" (Japanese)
- "I need to tailor my resume for a Product Manager role at Google"
- "Help me prepare resume versions in English and Chinese"

## 中文说明

### 这是什么？

一个 **Claude Skill（技能插件）**，让 Claude 变成一位拥有 15 年以上经验的资深 HR 专家，引导你完成从简历评审到岗位定向优化的全流程，支持任意语言。

### 核心功能

- **多语言支持**：上传任意语言的简历，获得中文、英文、日文、德文、法文、韩文等多语言优化版本，每个版本都遵循对应市场的简历规范
- **专业 HR 评审**：从第一印象、亮点、不足、ATS 兼容性等维度给出结构化反馈，并提供具体的重写建议
- **岗位定向优化**：粘贴职位描述后，AI 会搜索该岗位/公司的成功简历案例，逐条提出修改建议，每一处修改都由你确认后才应用
- **多岗位循环投递**：一次会话中可优化多个目标岗位，每个岗位生成专属定制版本
- **诚信原则**：只增强和重塑你的真实经历，绝不编造虚假资历

### 工作流程

```
第一步：上传简历（PDF、DOCX、TXT 或直接粘贴文字）
        ↓
第二步：获得专业评审 + 选择输出语言
        → 收到各语言的优化版 .docx 文件
        ↓
第三步：（可选）输入目标岗位信息
        → AI 搜索成功案例
        → 逐条提出修改建议供你确认
        → 生成定向优化的 .docx 文件
        ↓
第四步：继续为其他岗位重复第三步
```

### 安装方法

1. 下载本仓库中的 `resume-optimizer.skill` 文件
2. 在 Claude.ai 中进入 **设置 → Skills（技能）**
3. 上传 `.skill` 文件
4. 开始新对话，说："帮我优化简历"
